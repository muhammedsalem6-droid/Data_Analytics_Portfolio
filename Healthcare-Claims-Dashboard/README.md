# Healthcare Claims Performance Dashboard

## Business Problem
Healthcare insurers need visibility into claims cost, approval efficiency, and utilization
to support data-driven decision-making.

## Tools Used
- Power BI
- DAX
- Data Modeling

## KPIs
- Total Claims
- Total Cost
- Average Claim Cost
- Approval Rate

## Key Insights
- 85% approval rate indicates efficient claims processing
- Neurology and Cardiology are top cost-driving departments
- Claims peak between March and May
- Stroke and imaging procedures generate high costs

## Deliverables
- Power BI Dashboard
- Executive Summary Page
- Portfolio-ready PDF
